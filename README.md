# Teachers' Day Card

A special digital card to celebrate **Teachers' Day** and show appreciation for educators. This project can be used to send heartfelt messages to teachers or as a fun, interactive greeting card.

---

## Features

- Beautiful, customizable design
- Space for personalized messages
- Interactive animations (optional)
- Mobile and desktop friendly
- Easy to share via email or social media
